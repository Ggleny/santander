Installation
-------------
### From this git reposity
- Take git clone 
```
git clone 
```
#### Run manually
- Install dependencies
```
npm install
```
- Then run
```
npm start
```
#### Run with docker
- Start container
```
docker-compose up --build 
```
- Stop container
```
docker-compose stop 
```
- Remove container
```
docker-compose down 
```

Testing
-------------
### Run test
```
npm test 
```
